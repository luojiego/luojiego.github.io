学习目标

误区 从业7年以来，前5年都在使用 C++, 14年就开始接触 Redis 和 Protobuf, 但是到现在，依然只停留在使用层面，甚至只会使用一些简单的东西。类似于 Redis 中的集群，根本连了解都没有了解过，批量请求也没有使用过。接触 Go 以来，也是只会用, 原理不了解。 改进 每个疑惑点，最好深究原因； 重新学习，学而时习之，不亦乐乎； 学习源码，刚开始必然很难，可以从别人的讲解开始； 学习算法，之前学习的太浅，根本不能在大脑中停留太长时间； 坚持，每天早晨6点30起床，早睡早起，使用早起的时间来学习；
