财商

提高自己的财商 - 《富爸爸穷爸爸》读书笔记

正如作者提到的那样，财商在学校是不教授的。所以大多数人一生都活在对于钱的恐惧之中，这种恐惧让他们碌碌无为，跳不出作者提到的“老鼠赛跑”的陷阱。 书中好多话可以让我受用一生，例如：工作只是试图用暂时的办法来解决长期的问题、富人获得资产，而穷人和中产阶级获得负债，只不过他们以为那些负债就是资产、工人付出最大努力以免于被解雇，而雇主提供最低工资以防止工人辞职。 工作了十多年，虽说没有挣到多少钱，但是好在媳妇对于金钱的管理比较好。从来不买奢侈品，花钱比较有节制，养育着两个孩子，在最合适的时机买了房子，并且在有了积蓄之后，直接还清了所有贷款。没有车，几乎是没有负债的状态。 现在这个时间节点，我仍然比较迷茫。互联网泡沫的确比较明显，而且我也鼓起勇气离开了稳定的环境，去尝试新的可能性。尝试不对种类的产品确实比较有新鲜感，但是西安着实没有靠谱的游戏公司，并且目前游戏市场也不容乐观。未来的确充满了不确定性，目前唯一能做的是，就是积极学习，没有合适的职业，那投资自己就是最正确的选择。 财商这个关键词，我也是今年第一次听说，未来每年都需要去学习相关的知识，以弥补我在这方面的不足。 ...
